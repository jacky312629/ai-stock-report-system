# 📊 AI Stock Report System

自動化美股盤前策略報告系統  
結合市場數據抓取、AI分析與自動化報表生成

---

## 🚀 專案介紹

本專案為一套完整的自動化股票分析系統，  
每日自動抓取市場數據，並透過 AI 進行分析，  
生成可讀性高的 HTML 報表並自動寄送 Email。

👉 目標：打造「可實際使用」的決策輔助工具，而非單純技術展示

---

## 🧠 核心功能

### 📈 市場數據分析
- 抓取 NASDAQ、費城半導體、VIX、美元指數等指標
- 自動計算市場趨勢與風險狀態

### ⚠️ 風險控制機制
- 資料缺失自動判斷（避免 nan 錯誤）
- 資料不足時停止策略輸出（防止錯誤決策）
- 策略與市場狀態保持一致性

### 🤖 AI 分析
- 自動生成市場分析與操作建議
- 將複雜資訊轉為可讀性高的內容

### 📊 HTML 報表
- 視覺化市場儀表板
- 策略建議區塊
- 精選個股與新聞整理

### 📧 自動寄送
- 使用 Gmail API 自動寄送報表
- 支援排程（cron）

---

## 🛠️ 技術架構

- Python
- yfinance（市場數據）
- BeautifulSoup（新聞爬蟲）
- OpenAI API（AI分析）
- Gmail API（自動寄信）
- HTML/CSS（報表呈現）
